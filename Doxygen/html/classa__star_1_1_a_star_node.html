Detailed Description

This class provides the basic outline for implementing a graph structure, which can be used by the PathingAlgorithim.java aStarPath() method. By extending this class an providing correct functionality for the methods, this class will serve as a directed graph capable of finding the shortest path between 2 nodes

abstract float a_star.AStarNode.calculateHScore ( AStarNode  end)
pure virtual

Given the destination node, this method returns a "guess" value of the distance between this node and the end node. This can be the manhatten distance (s.x - e.x) + (s.y - e.y) + ... or the as the bird flies distance, so long as it is consistent. Note that this doesn't actually improve pathing, but a better implementation makes the algorithm run faster
