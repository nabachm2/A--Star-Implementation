float a_star.example.AStarMesh.calculateHScore ( AStarNode  end)
virtual

Given the destination node, this method returns a "guess" value of the distance between this node and the end node. This can be the manhatten distance (s.x - e.x) + (s.y - e.y) + ... or the as the bird flies distance, so long as it is consistent. Note that this doesn't actually improve pathing, but a better implementation makes the algorithm run faster

Parameters
endthe final goal node
Returns
the guess distance to an end node
